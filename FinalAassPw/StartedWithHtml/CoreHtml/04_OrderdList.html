Ordered List of HTML Tags

  1. <html>: The <html> tag is the root element of an HTML document. It wraps all the content on the page and indicates that the content is HTML.
  2. <head>: The <head> tag contains meta-information about the document, such as the title, character set, styles, and links to external resources. It is placed within the <html> tag.
  3. <title>: The <title> tag defines the title of the document, which appears in the browser's title bar or tab. It is placed inside the <head> tag.
  4. <body>: The <body> tag contains the content of the document that is visible to the user, including text, images, links, and other media. It is placed inside the <html> tag.
  5. <h1> to <h6>: These tags are used to define headings, with <h1> being the highest level (most important) heading and <h6> the lowest level (least important) heading. Headings are used to structure the content and improve accessibility.
  6. <p>: The <p> tag defines a paragraph of text. It is a block-level element that typically contains a chunk of text content, and it adds space above and below the paragraph by default.
  7. <a>: The <a> tag defines a hyperlink, which is used to link to another page, section, or external resource. The destination URL is specified using the href attribute.
  8. <img>: The <img> tag is used to embed an image into the document. It is a self-closing tag that requires the src attribute to specify the path to the image and the alt attribute to provide alternative text for accessibility.
  9. <ul>: The <ul> tag defines an unordered (bulleted) list. Each list item is enclosed within an <li> tag.
  10. <ol>: The <ol> tag defines an ordered (numbered) list. Each list item is enclosed within an <li> tag.
  11. <li>: The <li> tag defines a list item and is used within <ul> (unordered list) or <ol> (ordered list) tags.
  12. <div>: The <div> tag is a block-level element used to group content together. It is often used for layout purposes and can contain other HTML elements such as text, images, and forms.
  13. <span>: The <span> tag is an inline element used to group text or other inline elements. It is commonly used to apply styles to a specific part of the text.
  14. <form>: The <form> tag is used to create an HTML form for user input. It contains form elements such as input fields, checkboxes, radio buttons, and submit buttons.
  15. <input>: The <input> tag is used to create interactive controls in a form, such as text fields, checkboxes, and buttons. It is a self-closing tag with various attributes like type and name to define its behavior.
  16. <table>: The <table> tag is used to create a table in an HTML document. It contains rows (<tr>) and cells (<td> for data cells and <th> for header cells).
  17. <th>: The <th> tag defines a header cell in a table. It is typically used within a <tr> (table row) element and is usually bold and centered by default.
  18. <td>: The <td> tag defines a standard data cell in a table. It is used within a <tr> (table row) element and can contain text, images, or other HTML content.
  19. <br>: The <br> tag inserts a line break in the text, creating a new line without starting a new paragraph. It is a self-closing tag.
  20. <hr>: The <hr> tag creates a horizontal line (or "rule") across the page, typically used to separate content sections. It is a self-closing tag.
